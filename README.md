# L2overSSH
Forwarding an L2 tunnel through an SSH connection on FreeBSD systems.
Довольно часто возникает потребность в доступе к сетевой инфраструктуре на уровне L2.
Идея не новая, но широкого распостранения не имеет.
Используюя связку OpenSSH и Netgraph реализуем это на практике

FreeBSD(client)-ssh-->tap0 --> tap0-sshd-FreeBSD(server)

1. Настройка sshd.
1.1 Организовываем SSH доступ по ключу.
1.2 На сервере правим sshd_config установив значения: PermitTunnel yes , UseDNS no

2. Настройка тунелей на сервере и клиенте.
2.1 С помощью Netgraph навешиваем vlan на интерфейсы клиента и сервера.
2.2 На клиенте рописываем задание в cron для создания и проверки тунеля.

3.2 Модификация созданых vlan
3.3 Удаление vlan
